# Web Development project


# Main goal

Use all the knowledge obtained through the web development course to make your own portfolio page, this will showcase your hability to design web pages.

# Requirements

## 1. The website must be written **without the use of libraries or frameworks** you must strictly use pure HTML, CSS and JavaScript.

## 2. The website have to have the next sections:

### 2.1. Home page (index.html)

the purpose is welcome visitors and guide them, must have:

- Hero area with your name, tagline, and photo

- Quick intro (2-3 sentences about your journey)

- Featured project highlight (your best work)

- "Explore My Work" button linking to Projects page

- "Get to Know Me" button linking to About page

### 2.2. Projects (projects.html)

Showcase your work in detail, quality over quantity - 4-6 solid projects is perfect, for each project it must have:

- Project title and tagline

- Screenshot/gif of the project

- Tech stack icons

- Project description (problem â†’ solution format)

- "Key Learnings" 

- "View My Code" section linking to GitHub profile (if possible)

### 2.3. Learning Journey (journey.html)

Show your growth and educational path 

- Timeline visualization (vertical or horizontal)

- Milestones: Started coding (when/what), First completed project, Key courses completed, Hackathons participated in, Current learning goals etc.

- Education section: University/college info, Relevant coursework, Academic achievements (if any)

- Certifications section

- Current Focus area (what you're learning now)

### 2.4. Skills & Tools (skills.html)

Demonstrate your technical capabilities, create a skills dashboard, where you show which tools and concepts you handle.


### 2.5. About Me (about.html)

Show your personality and story

- Your story (paragraph about how you got into your carreer)

- Personal photo (casual, friendly)

- Personal section: Hobbies and interests, Favorite books/personalities, Something unique about you, 

- Why I Love my Carrer (passion statement) 

- Goals & Aspirations (internship desires, dream projects)

- Fun facts (e.g., "I've built 10 projects in the last 3 months")

### 2.6. Contact & Connect (contact.html)

Make outreach easy and professional

- Contact form with: Name, email, message fields, Dropdown for inquiry type (Internship, Collaboration, Question)

- Direct contact info: Email, Other relevant socials

- "Let's Work Together" section: Internship availability, Project collaboration interest, Networking invitation 

## 3. All the pages must have consistent header/footer

[Home] | [Projects] | [Journey] | [Skills] | [About] | [Contact]

## 5. Footer on every page:

Copyright notice, quick links to all pages, social media icons with links, back to top button.
