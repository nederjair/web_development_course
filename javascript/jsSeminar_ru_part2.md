# Лабораторная работа по JavaScript

# ГЛАВНАЯ ЦЕЛЬ

Игра, реализованная в предыдущей лабораторной работе, — очень простая, но важная основа для понимания и практики работы JavaScript. С другой стороны, можно внести следующие улучшения:

1. Улучшенная вёрстка с помощью `css`

2. Улучшенная общая цветовая схема с помощью `css`

3. Улучшенный более крупный шрифт с помощью `css`

4. Кнопка сброса (reset)

5. Функция, проверяющая, выиграл ли пользователь

6. Счётчик попыток с ограничением количества попыток, зависящим от длины слова

7. Начальная подсказка, добавляющая определённый процент букв

8. Подсказка, добавляющая изображение

9. При выигрыше пользователя выводится сообщение и изображение

10. При проигрыше пользователя выводится сообщение и изображение

11. Измените игру так, чтобы все буквы слова отображались в беспорядке, а пользователю нужно было их упорядочить, чтобы составить слово и выиграть. (**необязательно**)

2. Реализуйте 10 предложенных улучшений, 11-е **необязательно**.
